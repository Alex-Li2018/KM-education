# KM-education
成都柯明教育官网

**柯明教育官网,(express+vue).**

## Features

  - node.js的后台
  - vue的前台

## Set up

```bash
$ npm install
// 运行前台项目
$ cd app

$ npm run dev 

// 运行后台
$ cd service

$ npm run dev
```
